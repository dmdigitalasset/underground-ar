# UNDERGROUND — AR Menu (Demo)

Web-based AR viewer untuk menu F&B UNDERGROUND. Repo TERPISAH dari
aplikasi utama (by design): aplikasi hanya menyimpan URL di kolom
`ar_url`, file 3D + viewer hidup di sini.

- `viewer.html?model=<key>` — halaman 3D + AR per item
- `models/` — file `.glb` (Android Scene Viewer) + `.usdz` (iOS Quick Look)
- Hosting: GitHub Pages

Model demo: pizza · steak-potatoes · purple-cake · choc-croissant

> Sementara untuk demo — akan digantikan repo project AR resmi
> (aplikasi cukup ganti URL di admin, nol perubahan kode).
